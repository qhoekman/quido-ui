## Installation

The Radio Card component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Radio Card component consists of four parts:

- **RadioCard** - The root component that manages the radio group state
- **RadioCardItem** - An individual radio card option
- **RadioCardItemLabel** - The label for a radio card item
- **RadioCardItemDescription** - Optional description text for a radio card item
