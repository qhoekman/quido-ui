# Chat Message

A component for displaying chat messages with avatar, user name, message content, and metadata.

<Meta of={ChatMessageStories} />
<Canvas of={ChatMessageStories.Default} />
<Controls />

## Installation

The Chat Message component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Chat Message component consists of multiple parts:

- **ChatMessage** - The root component that manages the layout and variant (sender/receiver)
- **ChatMessageUser** - Displays the user name
- **ChatMessageBubble** - The message bubble container
- **ChatMessageMeta** - Container for metadata (like timestamp)
- **ChatMessageBubbleText** - Text component for message content
- **ChatMessageMetaText** - Text component for metadata
