# Hover Card

A component that displays additional content when hovering over a trigger element.

## Usage

<Canvas of={HoverCardStories.Default} />

<Controls />

## Installation

The Hover Card component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Hover Card component consists of three parts:

- **HoverCard** - The root component that manages the open/closed state
- **HoverCardTrigger** - The element that triggers the hover card on hover/focus
- **HoverCardContent** - The content that is displayed when hovering
