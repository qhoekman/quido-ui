# Dropdown

A dropdown menu component that displays a list of options or actions in a popover overlay.

## Usage

<Canvas of={DropdownStories.Default} />

<Controls />

## Installation

The Dropdown component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Dropdown component consists of multiple parts:

- **Dropdown** - The root component that manages the open/closed state
- **DropdownTrigger** - The button that opens the dropdown
- **DropdownContent** - The content container (includes portal)
- **DropdownItem** - Individual menu item
- **DropdownLabel** - Label/heading for menu sections
- **DropdownSeparator** - Visual separator between menu sections
- **DropdownShortcut** - Keyboard shortcut display component
- **DropdownGroup** - Group container for related items
- **DropdownSub** - Submenu root component
- **DropdownSubTrigger** - Trigger for submenu items
- **DropdownSubContent** - Submenu content container
- **DropdownCheckboxItem** - Menu item with checkbox indicator
- **DropdownRadioGroup** - Radio group container
- **DropdownRadioItem** - Menu item with radio indicator
- **DropdownPortal** - Portal component for rendering dropdown content
