The `qui-tab-navigation` component is a component that displays a list of links in a tab navigation style.

## Installation

The Tab Navigation component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Tab Navigation component consists of nine parts:

- **TabNavigation** - The root component that provides the navigation container
- **TabNavigationItem** - Container for individual navigation items
- **TabNavigationLink** - The link/button component with variants (link, border, pill)
- **TabNavigationIndicator** - Indicator showing caret up/down for dropdown menus
- **TabNavigationRail** - Decorative rail element
- **TabNavigationMenu** - Dropdown menu container
- **TabNavigationMenugroup** - Group within menu
- **TabNavigationMenuitem** - Individual menu item
- **TabNavigationSeparator** - Separator in menu
