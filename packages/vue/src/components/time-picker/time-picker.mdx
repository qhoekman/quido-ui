A time picker component that allows users to select hours and minutes from a dropdown interface.

## Installation

The Time Picker component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Time Picker component consists of three parts:

- **TimePicker** - The root component that manages the open/closed state and displays the selected time
- **TimeSelect** - The time selection component with hour and minute columns
- **Input** - The input component used to display the selected time (already included in the project)
