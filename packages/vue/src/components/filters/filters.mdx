# Filters

A set of components for building filter interfaces with persistent state management via URL hash.

## Installation

The Filters component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Filters component consists of six parts:

- **FiltersForm** - The container form component
- **FilterCheckbox** - A checkbox filter item that syncs with the filter store
- **FilterRange** - A range slider filter for numeric ranges
- **FilterRadioGroup** - A radio group container for filter options
- **FilterRadioGroupItem** - Individual radio items within a FilterRadioGroup
- **FilterClearButton** - A button to clear all active filters

## Details

The filters will be set via the hash in the URL for persistence. The filter state is managed through the `useFilterStore` composable, which syncs filter values with the URL hash. This allows filters to persist across page refreshes and be shareable via URL.
