# Tag Input

A component that allows users to enter and manage multiple tags. Tags can be added by typing comma-separated values or pressing Enter, and removed by clicking the remove button or pressing Backspace when the input is empty.

## Installation

The Tag Input component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Tag Input component depends on the following components:

- **Tag** - Used to display individual tags
- **Input** - Used for entering new tags
- **Button** - Used for the remove button
