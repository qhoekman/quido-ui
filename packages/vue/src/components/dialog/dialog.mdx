# Dialog

A dialog component that displays content in a modal overlay, typically used for forms, confirmations, or other important interactions.

## Installation

The Dialog component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Dialog component consists of multiple parts:

- **Dialog** - The root component that manages the open/closed state
- **DialogTrigger** - The button that opens the dialog
- **DialogContent** - The content container (includes overlay and portal)
- **DialogOverlay** - The backdrop overlay (included in DialogContent)
- **DialogClose** - Button to close the dialog (included in DialogContent)
- **DialogHeader** - Optional header section for title and description
- **DialogFooter** - Optional footer section for actions
- **DialogTitle** - Title text component (required for accessibility)
- **DialogDescription** - Description text component (optional)
