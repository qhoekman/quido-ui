# Select

A select component that allows users to choose a single option from a list of options.

## Installation

The Select component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Select component consists of multiple parts:

- **Select** - The root component that manages the value and open state
- **SelectTrigger** - The button that opens the select dropdown
- **SelectContent** - The dropdown content container
- **SelectValue** - Displays the selected value or placeholder
- **SelectItem** - Individual selectable options
- **SelectLabel** - Label for grouping options (optional)
- **SelectGroup** - Container for grouping options (optional)
- **SelectSeparator** - Visual separator between groups (optional)
