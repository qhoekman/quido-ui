# Autocomplete

A component that provides an input field with a dropdown list of suggestions that can be filtered as the user types.

## Installation

The Autocomplete component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Autocomplete component consists of six parts:

- **Autocomplete** - The root component that manages the open/closed state and selected option
- **AutocompleteTrigger** - The input field with chevron icon that triggers the dropdown
- **AutocompleteContent** - The container for the dropdown list
- **AutocompleteItem** - Individual option items in the dropdown
- **AutocompleteGroup** - A wrapper for grouping items (optional)
- **AutocompleteLabel** - A label component for grouping items (optional)
