# Accordion

A vertically stacked set of interactive headings that each reveal a section of content.

## Installation

The Accordion component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Accordion component consists of four parts:

- **Accordion** - The root component that manages the open/closed state
- **AccordionItem** - A single accordion item that contains a trigger and content
- **AccordionTrigger** - The button that toggles the accordion item
- **AccordionContent** - The content that is shown/hidden
