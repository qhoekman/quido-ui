# Mega Menu

A navigation menu component that displays large dropdown menus with rich content, perfect for complex navigation structures.

## Usage

<Canvas of={MegaMenuStories.Default} />

<Controls />

## Installation

The Mega Menu component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Mega Menu component consists of multiple parts:

- **MegaMenu** - The root component that manages the navigation menu state
- **MegaMenuList** - Container for menu items
- **MegaMenuItem** - Individual menu item container
- **MegaMenuTrigger** - Button that opens/closes the menu content
- **MegaMenuContent** - The dropdown content panel
- **MegaMenuViewport** - Viewport container for the content (automatically included in MegaMenu)
- **MegaMenuLink** - Simple link component for non-dropdown items
- **MegaMenuIndicator** - Visual indicator showing the active menu position
- **MegaMenuListItem** - Pre-styled list item with title and description
