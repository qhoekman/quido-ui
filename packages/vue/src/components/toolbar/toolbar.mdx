## Installation

The Toolbar component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Toolbar component consists of three parts:

- **Toolbar** - The root component that provides fixed positioning and safe area support
- **ToolbarContent** - The content container with flex layout
- **ToolbarLink** - A link component for toolbar items
