## Installation

The Tree component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Tree component consists of multiple parts:

- **Tree** - The root component that contains all tree items
- **TreeItem** - An individual tree item that can be expanded/collapsed
- **TreeGroup** - A container for nested tree items
- **TreeItemIcon** - A wrapper for icons within tree items
- **TreeItemLabel** - A wrapper for labels within tree items
- **TreeItemIndicator** - An internal component that shows expand/collapse indicators
