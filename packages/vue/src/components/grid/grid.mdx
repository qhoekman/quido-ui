# Grid Component

A flexible grid layout component that allows you to create responsive grid layouts with customizable columns and item spans.

## Installation

The Grid component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Grid component consists of two parts:

- **Grid** - The root component that manages the grid layout with configurable columns
- **GridItem** - Individual items within the grid that can span multiple columns
