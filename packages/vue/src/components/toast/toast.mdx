A toast component that displays temporary notifications to users.

## Installation

The Toast component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Toast component consists of six parts:

- **Toast** - The root component that wraps the toast content
- **ToastViewport** - The container that holds all toasts and positions them
- **ToastTitle** - The title text of the toast
- **ToastDescription** - The description text of the toast (optional)
- **ToastAction** - An action button within the toast (optional)
- **ToastClose** - The close button for dismissing the toast
