# Navigation Menu

A navigation menu component that provides a flexible layout for website navigation with brand, items, and actions sections.

## Installation

The Navigation Menu component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Navigation Menu component consists of six parts:

- **NavigationMenu** - The root component that provides the main navigation container
- **NavigationMenuBrand** - Container for the brand/logo section
- **NavigationMenuLogo** - The logo link component
- **NavigationMenuItems** - Container for navigation menu items
- **NavigationMenuLink** - Individual navigation link component
- **NavigationMenuActions** - Container for action buttons/icons
