## Props

- `currentDate` (Date, optional): The currently selected date. Defaults to today.
- `currentMonth` (Date, optional): The month to display. Defaults to the month of `currentDate`.

## Events

- `valueChange`: Emitted when a date is selected. The event payload is the selected `Date` object.

## Features

- Month navigation with previous/next buttons
- Highlights today's date
- Highlights selected date
- Shows dates from previous/next month in gray
- Keyboard navigation support (Enter/Space to select)
- Accessible with proper ARIA attributes
