A gallery component that displays images in a tiered layout with thumbnail navigation.

## Installation

The Tiered Gallery component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Tiered Gallery component consists of two parts:

- **TieredGallery** - The root component that manages the tab state
- **TieredGalleryImage** - The image component for displaying gallery images
