## Props

- **center** (`L.LatLngExpression`): The center of the map as a latitude and longitude array. Defaults to `[52.370216, 4.895168]` (Amsterdam).
- **zoom** (`number`): The zoom level of the map. Defaults to `13`.
- **options** (`L.MapOptions`): Additional Leaflet map options. Defaults to `{}`.
- **layers** (`L.Layer[]`): Array of Leaflet layers to add to the map. Defaults to `[]`.
- **markers** (`L.Marker[]`): Array of Leaflet markers to add to the map. Defaults to `[]`.

## Features

- Uses PDOK tile layer by default (Dutch mapping service)
- Supports custom layers and markers
- Reactive updates when props change
- Full Leaflet API support through options prop
