A flexible card component for displaying content with header, title, description, content, and footer sections.

## Installation

The Card component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Card component consists of six parts:

- **Card** - The root component that wraps all card content
- **CardHeader** - Container for the card title and description
- **CardTitle** - The main heading of the card
- **CardDescription** - Optional description text below the title
- **CardContent** - The main content area of the card
- **CardFooter** - Footer section typically used for actions
