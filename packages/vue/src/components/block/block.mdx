## Installation

The Block component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Block component consists of two parts:

- **Block** - The main container component that displays content sections
- **BlockTitle** - The title component that can be used above the block
