# Table

A table component for displaying structured data with header, body, footer, and caption sections.

## Installation

The Table component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Table component consists of eight parts:

- **Table** - The root component that wraps the entire table
- **TableHeader** - The header section (thead)
- **TableBody** - The body section (tbody)
- **TableFooter** - The footer section (tfoot)
- **TableRow** - A table row (tr)
- **TableHead** - A header cell (th)
- **TableCell** - A data cell (td)
- **TableCaption** - A caption for the table
