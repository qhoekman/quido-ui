# Empty State

A component for displaying empty states with icons, titles, descriptions, and action buttons.

<Meta of={EmptyStateStories} />
<Canvas of={EmptyStateStories.Default} />
<Controls />

## Installation

The Empty State component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Empty State component consists of five parts:

- **EmptyState** - The root component that manages the layout and size
- **EmptyStateIcon** - The icon container
- **EmptyStateTitle** - The title text
- **EmptyStateDescription** - The description text
- **EmptyStateActionGroup** - The container for action buttons
