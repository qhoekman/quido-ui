A component that allows users to select a color from a predefined set of options.

## Installation

The Color Selector component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Color Selector component consists of three parts:

- **ColorSelector** - The root component that manages the selected value
- **ColorSelectorItems** - A container for the color items
- **ColorSelectorItem** - Individual color selection items
