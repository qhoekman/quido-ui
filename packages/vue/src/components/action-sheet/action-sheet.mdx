# Action Sheet

A mobile-friendly action sheet component that slides up from the bottom of the screen, typically used for presenting multiple options or actions to users.

## Usage

<Canvas of={ActionSheetStories.Default} />

<Controls />

## Installation

The Action Sheet component uses Vue's composition API and Teleport for portal functionality. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Action Sheet component consists of multiple parts:

- **ActionSheet** - The root component that manages the open/closed state
- **ActionSheetTrigger** - The button that opens the action sheet
- **ActionSheetContent** - The content container that slides up from the bottom (includes overlay)
- **ActionSheetHeader** - Optional header section for title and description
- **ActionSheetFooter** - Optional footer section for actions
- **ActionSheetTitle** - Title text component
- **ActionSheetDescription** - Description text component
- **ActionSheetClose** - Button to close the action sheet
