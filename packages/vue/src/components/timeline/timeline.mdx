# Timeline

A component for displaying a chronological sequence of events or steps.

<Canvas of={TimelineStories.Default} />

<Controls />

## Installation

The Timeline component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Timeline component consists of multiple parts:

- **Timeline** - The root component (ul element)
- **TimelineItem** - Individual timeline item (li element)
- **TimelineConnector** - Container for bullet and separator
- **TimelineBullet** - The circular bullet/icon container
- **TimelineSeparator** - The vertical line between items
- **TimelineContent** - The content area for each item
- **TimelineTitle** - The title heading (h1-h6)
- **TimelineDescription** - The description text (p or span)
