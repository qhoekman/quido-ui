# Reel

A horizontal scrolling container component for displaying a series of items in a reel format.

## Usage

<Canvas of={ReelStories.Default} />

<Controls />

## Installation

The Reel component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Reel component consists of four parts:

- **Reel** - The root component that wraps everything
- **ReelContent** - The scrollable container that holds the items
- **ReelItem** - Individual items within the reel
- **ReelButton** - Navigation buttons for scrolling (prev/next)
