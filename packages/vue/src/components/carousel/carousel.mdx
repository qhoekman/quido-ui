## Installation

The Carousel component consists of four parts:

- **Carousel** - The root component that manages the carousel state
- **CarouselContent** - The container that holds all carousel items
- **CarouselItem** - Individual items within the carousel
- **CarouselButton** - Navigation buttons for previous/next
