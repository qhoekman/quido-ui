A date picker component that allows users to select a date from a calendar popover.

## Installation

The Date Picker component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Date Picker component depends on the following components:

- **Input** - The input field that displays the selected date
- **Calendar** - The calendar component used for date selection
- **Popover** - The popover container (provided by Reka UI)
