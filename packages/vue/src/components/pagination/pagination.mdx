# Pagination

A component for navigating through pages of content.

## Usage

<Canvas of={PaginationStories.Default} />

## Installation

The Pagination component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Pagination component consists of seven parts:

- **Pagination** - The root navigation element that wraps all pagination content
- **PaginationContent** - The unordered list container for pagination items
- **PaginationItem** - A list item wrapper for pagination elements
- **PaginationLink** - A link element for page numbers with active state support
- **PaginationPrevious** - A link element for navigating to the previous page
- **PaginationNext** - A link element for navigating to the next page
- **PaginationEllipsis** - A visual indicator for skipped pages
