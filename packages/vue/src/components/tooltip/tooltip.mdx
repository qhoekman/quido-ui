A tooltip component that displays helpful information when hovering over an element.

## Installation

The Tooltip component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Tooltip component consists of five parts:

- **TooltipProvider** - The provider component that wraps your application (optional but recommended)
- **Tooltip** - The root component that manages the open/closed state
- **TooltipTrigger** - The element that triggers the tooltip
- **TooltipContent** - The content that is displayed in the tooltip
- **TooltipArrow** - An optional arrow pointing to the trigger
