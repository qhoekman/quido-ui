# Color Wheel

A component that displays an interactive color wheel for selecting colors by hue and saturation.

<Meta of={ColorWheelStories} />
<Canvas of={ColorWheelStories.Default} />

## Usage

The Color Wheel component displays a circular color picker where users can select colors by clicking or dragging on the wheel. The outer edge represents full saturation, while the center represents no saturation (grayscale). The hue changes as you move around the circle.

## Props

- `hue` (number): The hue value in degrees (0-360). Defaults to `0`.
- `saturation` (number): The saturation value as a percentage (0-100). Defaults to `100`.
- `lightness` (number): The lightness value as a percentage (0-100). Defaults to `50`.

## Events

- `valueChange`: Emitted when a color is selected. The event payload is a color value string in HSL format (e.g., `hsl(180, 80%, 50%)`).
