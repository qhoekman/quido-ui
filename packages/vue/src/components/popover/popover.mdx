# Popover

A popover component that displays content in a floating overlay triggered by a button or other element.

## Installation

The Popover component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Popover component consists of three parts:

- **Popover** - The root component that manages the open/closed state
- **PopoverTrigger** - The button that opens/closes the popover
- **PopoverContent** - The content that is displayed in the popover
