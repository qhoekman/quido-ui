## Props

- `value` (string): The password input to evaluate strength
- `requirements` (Requirement[]): Array of password requirements with regex patterns and descriptions
- `thresholds` ([number, number]): Thresholds for determining strength levels [min, max]

## Strength Levels

- **Weak**: When the number of met requirements is less than or equal to the first threshold
- **Medium**: When the number of met requirements is between the thresholds
- **Strong**: When the number of met requirements exceeds the second threshold
