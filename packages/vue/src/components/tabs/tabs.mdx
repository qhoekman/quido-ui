A set of layered sections of content—known as tab panels—that are displayed one at a time.

## Installation

The Tabs component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Tabs component consists of four parts:

- **Tabs** - The root component that manages the active tab state
- **TabsList** - Container for the tab triggers
- **TabsTrigger** - Button that activates a tab panel
- **TabsContent** - Content panel that is shown when its trigger is active
