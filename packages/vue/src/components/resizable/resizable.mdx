A component for creating resizable panel layouts with drag handles.

## Installation

The Resizable component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Resizable component consists of three parts:

- **ResizablePanelGroup** - The container that manages the layout direction
- **ResizablePanel** - Individual panels that can be resized
- **ResizableHandle** - The drag handle between panels
