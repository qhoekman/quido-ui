## Installation

The Toggle Group component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Toggle Group component consists of two parts:

- **ToggleGroup** - The root component that manages the selected values
- **ToggleGroupItem** - Individual toggle items within the group
