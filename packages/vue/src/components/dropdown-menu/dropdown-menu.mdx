# Dropdown Menu

A dropdown menu component that displays a list of options or actions in a popover overlay.

## Usage

<Canvas of={DropdownMenuStories.Default} />

<Controls />

## Installation

The Dropdown Menu component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Dropdown Menu component consists of multiple parts:

- **DropdownMenu** - The root component that manages the open/closed state
- **DropdownMenuTrigger** - The button that opens the dropdown
- **DropdownMenuContent** - The content container (includes portal)
- **DropdownMenuItem** - Individual menu item
- **DropdownMenuLabel** - Label/heading for menu sections
- **DropdownMenuSeparator** - Visual separator between menu sections
- **DropdownMenuShortcut** - Keyboard shortcut display component
- **DropdownMenuGroup** - Group container for related items
- **DropdownMenuSub** - Submenu root component
- **DropdownMenuSubTrigger** - Trigger for submenu items
- **DropdownMenuSubContent** - Submenu content container
- **DropdownMenuCheckboxItem** - Menu item with checkbox indicator
- **DropdownMenuRadioGroup** - Radio group container
- **DropdownMenuRadioItem** - Menu item with radio indicator
- **DropdownMenuPortal** - Portal component for rendering dropdown content
