# Color Select

A component that allows users to select a color from a set of predefined color options displayed as circular color indicators.

## Usage

The Color Select component consists of two parts:

- **ColorSelect** - The root component that manages the selected color state
- **ColorSelectItem** - Individual color options displayed as circular buttons

## Installation

The Color Select component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Props

### ColorSelect Props

- `value` (string | undefined): Controlled value for the selected color
- `defaultValue` (string | undefined): Default value for the selected color (uncontrolled mode)
- `disabled` (boolean): Whether the color select is disabled
- `asChild` (boolean): Whether to render as a child element
- `as` (string): The element type to render (default: 'div')

### ColorSelectItem Props

- `color` (string): The color value to display (CSS variable or hex color)
- `asChild` (boolean): Whether to render as a child element
- `as` (string): The element type to render (default: 'button')

## Events

### ColorSelect Events

- `update:value`: Emitted when the selected color changes (for v-model support)
- `valueChange`: Emitted when the selected color changes

## Controlled vs Uncontrolled

The Color Select component supports both controlled and uncontrolled usage:

**Controlled (with v-model):**
```vue
<ColorSelect v-model="selectedColor">
  <ColorSelectItem color="var(--color-red-500)" />
  <ColorSelectItem color="var(--color-green-500)" />
</ColorSelect>
```

**Uncontrolled (with defaultValue):**
```vue
<ColorSelect default-value="var(--color-red-500)">
  <ColorSelectItem color="var(--color-red-500)" />
  <ColorSelectItem color="var(--color-green-500)" />
</ColorSelect>
```
