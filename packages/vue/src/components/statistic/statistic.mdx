# Statistic

A component for displaying statistics with labels, values, trends, and helper text.

<Canvas of={StatisticStories.Default} />

<Controls />

## Installation

The Statistic component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Statistic component consists of five parts:

- **Statistic** - The root component that manages spacing and layout
- **StatisticLabel** - The label text for the statistic
- **StatisticValue** - The main value display (typically a heading element)
- **StatisticTrend** - Optional trend indicator with up/down/neutral variants
- **StatisticHelperText** - Optional helper text below the statistic
