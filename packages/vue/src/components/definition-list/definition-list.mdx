A component for displaying term-description pairs in a structured list format.

## Installation

The Definition List component consists of four parts:

- **DefinitionList** - The root container (`dl` element)
- **DefinitionListItem** - Wrapper for each term/description pair (`div` element)
- **DefinitionTerm** - The term/label (`dt` element)
- **DefinitionDescription** - The description/value (`dd` element)
