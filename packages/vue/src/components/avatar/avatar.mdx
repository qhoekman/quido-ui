## Installation

The Avatar component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Avatar component consists of three parts:

- **Avatar** - The root component that manages the avatar state
- **AvatarImage** - The image component that displays the avatar image
- **AvatarFallback** - The fallback component that displays when the image fails to load
