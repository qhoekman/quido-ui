## Features

- Automatically shows banner if no consent has been given
- Stores consent preference in localStorage
- Provides Accept and Reject actions
- Includes close button to dismiss banner
- Responsive design with fixed positioning

## Behavior

The component checks localStorage for a `cookieConsent` key on mount. If no consent value is found, the banner is displayed. When the user clicks "Accept" or "Reject", the preference is saved to localStorage and the banner is hidden.
