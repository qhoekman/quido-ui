# Combobox

A component that provides a button trigger with a dropdown list of options that can be filtered with a search input.

## Usage

<Controls />

## Installation

The Combobox component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Combobox component consists of eight parts:

- **Combobox** - The root component that manages the open/closed state and selected option
- **ComboboxTrigger** - The button that displays the selected value or placeholder and triggers the dropdown
- **ComboboxContent** - The container for the dropdown list
- **ComboboxItem** - Individual option items in the dropdown
- **ComboboxGroup** - A wrapper for grouping items (optional)
- **ComboboxLabel** - A label component for grouping items (optional)
- **ComboboxSearch** - A search input component (optional)
- **ComboboxSearchbox** - A wrapper for the search input with icon (optional)
