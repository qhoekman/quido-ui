# Navbar

A navigation bar component for building website headers with navigation links, dropdowns, and action buttons.

<Canvas of={NavbarStories.Default} />

<Controls />

## Installation

The Navbar component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Navbar component consists of multiple parts:

- **Navbar** - The root component that wraps the navigation
- **NavbarBrand** - Container for the brand/logo section
- **NavbarLogo** - Logo link component
- **NavbarItems** - Container for navigation items
- **NavbarLink** - Navigation link component
- **NavbarDropdown** - Dropdown menu wrapper
- **NavbarDropdownTrigger** - Trigger button for dropdown
- **NavbarDropdownContent** - Dropdown content container
- **NavbarDropdownLink** - Link inside dropdown menu
- **NavbarActions** - Container for action buttons
