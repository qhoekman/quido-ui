# Footer

A comprehensive footer component system for displaying site navigation, links, and copyright information.

<Canvas of={FooterStories.Default} />

<Controls />

## Installation

The Footer component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Footer component consists of multiple parts:

- **Footer** - The root component that wraps all footer content
- **FooterContent** - Container for the main footer content
- **FooterColumns** - Grid container for footer columns
- **FooterColumn** - Individual column container
- **FooterHeading** - Heading for footer sections
- **FooterColumnList** - List container for column links
- **FooterColumnListItem** - Individual list item in a column
- **FooterSection** - Side section (e.g., newsletter signup)
- **FooterBanner** - Bottom banner with copyright and links
- **FooterBannerList** - Container for banner links
- **FooterBannerListItem** - Individual item in banner list
- **FooterCopyright** - Copyright text component
- **FooterLink** - Styled link component for footer
