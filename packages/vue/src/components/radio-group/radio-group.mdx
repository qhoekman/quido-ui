# Radio Group

A set of checkable buttons—known as radio buttons—where no more than one of the buttons can be checked at a time.

## Installation

The Radio Group component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Radio Group component consists of three parts:

- **RadioGroup** - The root component that manages the selected value
- **RadioGroupItem** - An individual radio button item
- **RadioGroupItemIndicator** - The visual indicator shown when an item is selected (optional, included by default in RadioGroupItem)
