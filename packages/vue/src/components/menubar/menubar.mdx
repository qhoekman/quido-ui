# Menubar

A horizontal menu bar component that displays a set of menu triggers, each opening a dropdown menu when clicked.

## Usage

<Controls />

## Installation

The Menubar component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Menubar component consists of multiple parts:

- **Menubar** - The root container component
- **MenubarTrigger** - The button that opens/closes a menu
- **MenubarMenu** - The dropdown menu content panel
- **MenubarGroup** - Container for grouping menu items
- **MenubarItem** - Individual menu item
- **MenubarSeparator** - Visual separator between menu groups
