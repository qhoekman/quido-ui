# Breadcrumb

A navigation component that displays the current page's location within a site's hierarchy.

## Installation

The Breadcrumb component uses Reka UI primitives under the hood. Make sure you have `reka-ui` installed:

\`\`\`bash
npm install reka-ui
\`\`\`

## Components

The Breadcrumb component consists of multiple parts:

- **Breadcrumb** - The root navigation component
- **BreadcrumbList** - The ordered list wrapper
- **BreadcrumbItem** - Individual breadcrumb item wrapper
- **BreadcrumbSeparator** - Separator between items (defaults to slash icon)
- **BreadcrumbLink** - Link component for navigable items
- **BreadcrumbPage** - Current page indicator
- **BreadcrumbEllipsis** - Overflow indicator for collapsed items
